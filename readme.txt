To play this game, you have 2 paths:

## First path:
1. Download this project by clicking on Code -> Download ZIP
2. Unzip folder and open it in PyCharm
3. Open "main.py" file and run it. 
## Second path:
- Go to https://replit.com/@renatocrovellad/guess-italy-game?v=1 and hit "run".


Also, there are two ways to play this game: 
1. Guess Italy regions
  If you want to test your general knowledgement of Italy's geography, you can play this quiz by typing "Italy" in first prompt screen. You'll have to guess all Italy's regions in order to win! TIP: they're all in Italian.
2. Guess Sicily's provinces
  If you're a big fan of Sicily (like me) and want to test your knowledge about this specific region, you should type "Sicily" in first prompt screen. Would you be able to guess all 9 provinces of Sicily in order to win this game? TIP: their names are all in Italian.
